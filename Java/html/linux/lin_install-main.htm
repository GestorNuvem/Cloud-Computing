Instalação do Software
para instalação em Linux

Clique no botão de título de cada software na janela à esquerda para ler as instruções de instalação. Embora todo o software seja distribuído neste CD-ROM, você precisará estar conectado à Internet para poder acessar vários links importantes das instruções de instalação.

Preste muita atenção aos requisitos de software relativos aos programas necessários, que devem estar instalados antes de continuar com a instalação dos pacotes de software listados na janela à esquerda.

Requisitos do ambiente de software

Para instalar e usar vários dos programas incluídos neste CD-ROM, você já deve ter alguns softwares necessários instalados. Na maioria dos casos, este software é fornecido como um pacote de software separado ou como um componente de um dos pacotes. Alguns dos requisitos, no entanto, não estão incluídos no CD.

A tabela a seguir indica as dependências dos diversos programas de software entre eles e em relação a outros programas. Você deve ler esta tabela agora para obter uma visão geral e relê-la antes de instalar qualquer software deste CD-ROM. Volte a esta tabela novamente para depurar quaisquer problemas de instalação que possam surgir. Leia as anotações que fornecem mais detalhes sobre muitos destes requisitos. Em cada caso, é altamente recomendado que você veja a homepage dos dois programas para determinar o que mudou entre a gravação do CD e sua compra do livro-texto.
